# Cash Flow Dashboard

## 📌 Description
Cash Flow Dashboard is a web-based expense tracker that helps users manage their salary and expenses efficiently. It provides visual insights using Chart.js and stores data using Local Storage.

## 🚀 Features
- Add Expenses
- Delete Expenses
- Expense Categories
- Transaction Counter
- Remaining Balance Calculation
- Pie Chart Analytics
- Local Storage Support
- Responsive Design

## 🛠 Technologies Used
- HTML
- CSS
- JavaScript
- Chart.js
- Local Storage

## 📂 Project Structure
├── index.html
├── style.css
├── script.js
├── README.md
└── prompts.md

## ▶ How to Run
1. Download or clone the repository.
2. Open index.html in any browser.
